## 语法

```
@since <versionDescription>
```

## 概述

`@since` 标记表示在特定版本中添加了类、方法或其他符号。

## 实例

使用 `@since`:

```js
/**
 * Provides access to user information.
 * @since 1.0.1
 */
function UserRecord() {}
```
