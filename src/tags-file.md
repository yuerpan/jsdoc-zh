## 别名

```
@fileoverview
@overview
```

## 语法

```
@file <descriptionOfFile>
```


## 概述

`@file` 标签提供文件的说明。在文件开头的 JSDoc 注释部分使用该标签。

## 实例

文件描述：

```
/**
 * @file Manages the configuration settings for the widget.
 * @author Rowina Sanela 
 */
```
