# @todo

## 目录

- [语法](#语法)
- [概述](#概述)
- [实例](#实例)

## 语法

```
@todo text describing thing to do.
```

## 概述

`@todo` 标签记录要完成的任务。在一个 JSDoc 注释块中您可以包含多个 `@todo` 标签。

## 实例

使用 `@todo` 标签：

```js
/**
 * @todo Write the documentation.
 * @todo Implement this function.
 */
function foo() {
  // write me
}
```
